Thema dieser Bachelorarbeit
===============

Im Zuge der Bachelorarbeit sollen die Voraussetzungen geschaffen werden,
ein redundantes Dateisystem, dass direkt auf zwei oder mehr
NAND-Flashbänken arbeitet, als betriebssystem- und prozessorunabhängiges
Paket zu entwickeln. Dabei wird als Erstes ermittelt, welche
Anforderungen an Funktionalität und Hardware gestellt werden. Dabei
werden u.A. Faktoren wie tolerierte Fehlerklassen, also z.B. Abnutzung
einzelner *Pages*, *Single-Event-Upsets* / *latch-ups*, Geschwindigkeit
in Lese- und Schreibvorgängen, benötigte CPU-Zeit und Mindestgröße des
RAMs, berücksichtigt und deren Grenzen im Sinne der Anforderungen
gesetzt. Auf dieser Basis wird recherchiert, welche dafür infrage
kommenen Dateisysteme bereits veröffentlicht wurden, und nach ihrer
Tauglichkeit bewertet. Als praktischen Teil soll eine
Simulationsumgebung programmiert werden, das eine Schnittstelle bietet,
mit der genannte Dateisysteme auf die gefundenen Gütekriterien evaluiert
werden können. Dabei wird auch recherchiert, welche
Simulationsumgebungen bereits existieren, und evaluiert, wie gut sie für
diesen Zweck anwendbar sind. Im Zuge der Programmierung werden ein oder
mehrere existierende Dateisysteme mit diesem System simuliert und ihre
Performanz bewertet.


Dit's der Programmieranteil.
